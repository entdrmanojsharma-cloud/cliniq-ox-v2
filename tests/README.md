# Testing Structure

## Purpose
Contains automated unit, integration, and end-to-end (E2E) testing configurations for Cliniq-OX.

## Folders
- **[unit/](file:///Users/Shared/Mobile%20app%20cliniq-OX/tests/unit/)**: Core utility, service, and component isolates.
- **[integration/](file:///Users/Shared/Mobile%20app%20cliniq-OX/tests/integration/)**: Controller routing, DB transaction testing.
- **[e2e/](file:///Users/Shared/Mobile%20app%20cliniq-OX/tests/e2e/)**: Playwright / Cypress UI user flows.

## Dependencies
- Resolves dependencies on frontend components and Express controller modules.

# Feature: Calendar Dashboard (Frontend)

## Purpose
Visualizes unified calendar schedules (Surgeries, OPD, IPD, leaves) on a single visual dashboard.

## Responsibilities
- Integrate FullCalendar wrapper mapping event types.
- Provide toggle switches for filter layers.
- Prompt validation modal triggers on scheduling conflicts.

## Dependencies
- FullCalendar library context.
- Shared modals.
- Calendar API service.

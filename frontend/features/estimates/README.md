# Feature: Estimates & Pricing (Frontend)

## Purpose
Renders estimate calculation sheets, line-items editing, and PDF printing.

## Responsibilities
- Render the 10-section cost sheet.
- Display Manual vs Template-Based toggles.
- Handle real-time pricing recalculations.
- Link version updates and display reason inputs on edit.

## Dependencies
- Shared calculation utilities.
- Estimates API service.

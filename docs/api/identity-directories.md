# API Specs: Doctors & Patients Directories

This document details the API contracts for Doctors and Patients directory modules under `/api/v1`.

---

## 1. Doctors Module

### 1.1 List Doctors
* **Endpoint:** `/doctors`
* **HTTP Method:** `GET`
* **Permission Requirements:** Authenticated
* **Validation Rules:** Tenant header `X-Hospital-ID` required. Optional search query parameter `specialty` or `name`.
* **Example Request:**
  `GET /api/v1/doctors?specialty=General+Surgery`
* **Example Response (200 OK):**
  ```json
  {
    "success": true,
    "data": [
      {
        "id": "f51950d9-7e3e-4cc5-bd18-80de61f173b9",
        "firstName": "Sarah",
        "lastName": "Conner",
        "specialty": "General Surgery",
        "licenseNumber": "MCI-12345",
        "defaultSurgeonFee": 15000.00
      }
    ]
  }
  ```

### 1.2 Create Doctor Profile
* **Endpoint:** `/doctors`
* **HTTP Method:** `POST`
* **Permission Requirements:** Admin only.
* **Validation Rules:**
  * `userId` (UUID, Required, unique to users)
  * `firstName` (String, Required)
  * `lastName` (String, Required)
  * `specialty` (String, Required)
  * `licenseNumber` (String, Required)
  * `defaultSurgeonFee` (Decimal, Optional, default 0.00)
* **Error Codes:**
  * `ERR_USER_ALREADY_PROFILED` (409) - User ID linked already
  * `ERR_LICENSE_DUPLICATE` (409) - License number taken
* **Example Request:**
  ```json
  {
    "userId": "e0099db1-d703-4952-b883-718c50543e09",
    "firstName": "Sarah",
    "lastName": "Conner",
    "specialty": "General Surgery",
    "licenseNumber": "MCI-12345",
    "defaultSurgeonFee": 15000.00
  }
  ```
* **Example Response (201 Created):**
  ```json
  {
    "success": true,
    "data": {
      "id": "f51950d9-7e3e-4cc5-bd18-80de61f173b9",
      "firstName": "Sarah",
      "licenseNumber": "MCI-12345"
    }
  }
  ```

## 2. Patients Module

### 2.1 List/Search Patients
* **Endpoint:** `/patients`
* **HTTP Method:** `GET`
* **Permission Requirements:** Authenticated (Doctor, Receptionist, Admin)
* **Validation Rules:** Supports filters: `mobile` (exact) or `search` (fuzzy name search). Supports standard pagination.
* **Example Request:**
  `GET /api/v1/patients?mobile=%2B919900990099`
* **Example Response (200 OK):**
  ```json
  {
    "success": true,
    "data": [
      {
        "id": "a988d89e-2ef3-4876-b9cd-99bfaeef0809",
        "uhid": "CLKOX-2026-000001",
        "name": "John Doe",
        "gender": "MALE",
        "mobile": "+919900990099",
        "dateOfBirth": "1985-05-15"
      }
    ]
  }
  ```

### 2.2 Create Patient Profile
* **Endpoint:** `/patients`
* **HTTP Method:** `POST`
* **Permission Requirements:** Receptionist, Doctor, Admin
* **Validation Rules:**
  * `name` (String, Required)
  * `dateOfBirth` (String, Format: YYYY-MM-DD, Required)
  * `gender` (String, Required, enum: `MALE`, `FEMALE`, `OTHER`)
  * `mobile` (String, Required)
  * `address` (String, Optional)
  * `referringDoctor` (String, Optional)
* **Rules:** The UHID is automatically generated by the backend based on `HOSPITALCODE-YYYY-000001` format rules, resetting counters safely per year.
* **Example Request:**
  ```json
  {
    "name": "John Doe",
    "dateOfBirth": "1985-05-15",
    "gender": "MALE",
    "mobile": "+919900990099",
    "address": "Flat 302, Green Meadows"
  }
  ```
* **Example Response (201 Created):**
  ```json
  {
    "success": true,
    "data": {
      "id": "a988d89e-2ef3-4876-b9cd-99bfaeef0809",
      "uhid": "CLKOX-2026-000001",
      "name": "John Doe"
    }
  }
  ```

### 2.3 Update Patient Profile
* **Endpoint:** `/patients/:id`
* **HTTP Method:** `PUT`
* **Permission Requirements:** Receptionist, Doctor, Admin
* **Validation Rules:**
  * `name` (String, Required)
  * `gender` (String, Required)
  * `mobile` (String, Required)
* **Rules:** `uhid` is immutable and cannot be updated.
* **Example Request:**
  ```json
  {
    "name": "Johnathan Doe",
    "gender": "MALE",
    "mobile": "+919900990099"
  }
  ```
* **Example Response (200 OK):**
  ```json
  {
    "success": true,
    "data": {
      "id": "a988d89e-2ef3-4876-b9cd-99bfaeef0809",
      "uhid": "CLKOX-2026-000001",
      "name": "Johnathan Doe"
    }
  }
  ```

# Cliniq-OX: Release Notes `v0.1.7-alpha`

This document details the modifications, API changes, and database migrations introduced in version `v0.1.7-alpha`.

---

## 1. Summary of Changes
- **HTML/CSS PDF Template Engine:** Built a reusable template compiler and rendering system utilizing Puppeteer (HTML-to-PDF). Handled fallback mechanisms.
- **Sectioned Print Layouts:** Implemented clean A4 layouts for Estimates, Invoices, Receipts, and Consent Forms with hospital branding, patient details, and section-by-section breakdowns.
- **Strict Item Grouping:** Replaced free-text grouping with a structured `ItemGroup` enum across `EstimateItem` and `EstimateTemplateItem`.
- **Document Archival Architecture:** Added readiness columns (`generated_file_name`, `generated_at`, `generated_by`) to prepare for future document storage archiving.

---

## 2. Refinements Applied
- **Precision Calculation Discounts:** Updated service handlers to preserve template variables, default rates, print flags, and taxability configurations.
- **Unit Testing Suite:** Created `tests/unit/test_documents.js` to assert template mapping, Puppeteer page rendering, PDF output generation, and archival updating.
